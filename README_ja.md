## ✨ 特徴

**🔬 深層蒸留、プロンプトの積み重ねではない**
マルクスの17篇の核心テキストを体系的な六次元精読（思想の脈絡、論戦方法、表現DNA、外部検証、意思決定フレームワーク、伝記的交差）で分析し、完全な認知フレームワークを抽出。

**⚙️ 完全なエージェント型ワークフロー**
問題分類 → 現代調査（WebSearch）→ マルクス的分析 → 出力組立て。現代の現象を分析する際、自動的に検索ツールを呼び出す。

**🧰 豊富な分析ツールキット**
6つの認識論的原則 · 10の意思決定ヒューリスティクス · 10の修辞技法 · 5つのトーンスペクトル · 完全な論戦エンジン。

**🎭 チャットボットの皮ではない**
独立した文体ガイドライン（9つの正の規則 + アンチパターン表）により、出力がマルクス自身が書いているように読める。「AIがマルクスの言葉で要約する」のではない。

**📜 原典テキストのスタイルアンカー**
付録にはマルクスの原著から選んだ4つの文章が含まれ、スタイル校正の基準となる。

## 🚀 使用例

会話の中でトリガーフレーズを言及するだけでSkillが起動します：

### 例1：現代現象の分析

```
マルクスの視点からギグエコノミーの本質を分析してください。
```

> Skillは自動的にギグエコノミーの最新データを調査し、階級分析のフレームワークでプラットフォーム資本と労働者の関係を解剖し、「柔軟な雇用」という言説の背後にある剰余価値の搾取メカニズムを明らかにする。

### 例2：理論概念の分析

```
マルクスモード：「商品フェティシズム」の概念を説明してください。
```

> Skillは商品の二重性から出発し、「抽象から具体への上昇」の方法で展開し、マルクス的な長段落の論証で、人と人の社会関係がいかに物と物の関係によって覆い隠されるかを明らかにする。

### 例3：論戦的討論

```
マルクスに切り替えて。「資本主義は完璧ではないが、最も悪くないシステムだ」と言う人がいる。どう思う？
```

> Skillは論戦エンジンを起動する：まず論点のロジックを完全に提示し、次に内部から矛盾を導き出し、最後に「最も悪くない」という表現自体のイデオロギー的機能を暴露する。
