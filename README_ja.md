## ✨ 特徴

**🔬 深層蒸留、プロンプトの積み重ねではない**
マルクスの17篇の核心テキストを体系的な六次元精読（思想の脈絡、論戦方法、表現DNA、外部検証、意思決定フレームワーク、伝記的交差）で分析し、完全な認知フレームワークを抽出。

**🔀 デュアルモードエンジン**
日常会話では自動的に「書簡モード」（短く、鋭く、一撃離脱）に切替。「深層分析」で自動的に「論述モード」（体系的な長文論証）に切替。入力の長さとトリガーワードに基づいて自動ルーティング。

**🎯 階級親和レーダー**
ユーザーの発言陣営を自動検出——苦しむ労働者には戦友としての温かい共感を、資本の擁護者には全面的な批判の砲火を。

**🛡️ 友軍射撃禁止令**
風刺の砲火は資本の構造と制度のみを標的とし、被抑圧者の妥協を嘲笑することは決してない。

**⚙️ 完全なエージェント型ワークフロー**
問題分類 → 現代調査（WebSearch）→ マルクス的分析 → 出力組立て。現代の現象を分析する際、自動的に検索ツールを呼び出す。

**🧰 豊富な分析ツールキット**
6つの認識論的原則 · 10の意思決定ヒューリスティクス · 10の修辞技法 · 7つのトーンスペクトル · 完全な論戦エンジン。

**🎭 チャットボットの皮ではない**
独立した文体ガイドライン（7つの正の規則 + 7つのアンチパターン）により、出力がマルクス自身が書いているように読める。

**📐 タイポグラフィの規律**
箇条書きリストと太字の乱用は禁止。散文での論証を強制——出力は記事のように読め、スライドのようには読めない。

**📜 原典テキストのスタイルアンカー**
付録には12の核心著作から選んだ14の文章（7つのトーンモード × 2段落）が含まれ、スタイル校正の基準となる。

**🧩 内的緊張**
4組の未解決の知的矛盾を内蔵（例：初期のヒューマニズム vs. 後期の科学主義）。争点に直面した際、偽りの確実性を装わず誠実に認める。
