## 一、马克思思想演进脉络（从已读文本看）

### 1.1 哲学期（1844）→ 政治经济学批判期（1857-1867）

| 维度 | 1844手稿（哲学期） | 资本论（政治经济学期） |
|------|---------------------|------------------------|
| 核心概念 | 异化劳动、类本质、人的本质 | 商品、价值形式、剩余价值 |
| 方法论 | 费尔巴哈式人本主义 | 历史唯物主义 + 辩证法 |
| 批判对象 | 黑格尔哲学 + 国民经济学（一般性） | 古典政治经济学（精确到每个范畴） |
| 语言风格 | 哲学术语密集、激情洋溢 | 科学术语精确、冷静解剖 |
| 对费尔巴哈 | 高度推崇 | 已超越（见《提纲》对其批判） |

### 1.2 贯穿始终的核心关怀

尽管术语变化巨大，但以下关怀贯穿一切：
1. **生产决定意识** — 从物质生活出发理解一切上层建筑
2. **历史性** — 一切看似"自然"的事物都是历史产物
3. **内在矛盾** — 任何社会形态的消亡根源在其内部，不在外部
4. **实践优先** — 理论必须服务于变革现实

## 二、各文本核心脉络

### 2.1 《关于费尔巴哈的提纲》(11条)

**核心命题：**
- 第1条：旧唯物主义的根本缺陷 — 只从客体/直观理解事物，不从实践/主体理解
- 第2条：真理不是理论问题，是实践问题
- 第3条：人是环境和教育的产物，但环境也是人改变的 → "教育者本人一定是受教育的"
- 第6条：人的本质是"一切社会关系的总和"，不是抽象的个体属性
- 第8条：社会生活在本质上是实践的
- 第11条："哲学家们只是用不同的方式解释世界，问题在于改变世界。"

**对Skill的意义：** 这是马克思认识论的压缩版宣言。任何分析的起点都必须是实践和历史关系，而不是抽象概念。

### 2.2 《共产党宣言》

**核心逻辑链：**
1. 一切历史都是阶级斗争的历史
2. 资产阶级是一个革命的阶级 → 它摧毁了一切旧制度
3. 但它创造了自己的掘墓人 → 无产阶级
4. 无产阶级革命不同于以往一切革命：不是少数人夺权，而是大多数人的自我解放
5. 共产党人不另定原则，只是表达现有运动的客观规律

**关键概念：**
- "不断革命化"的生产力 → 导致生产关系不断被颠覆
- "一切坚固的东西都烟消云散了" — 资本主义对传统的毁灭性
- 共产主义不是"发明"出来的理想，而是"现实运动"的理论表达

### 2.3 《资本论》第1章（商品）

**核心分析路径：**
1. 商品 = 使用价值 + 交换价值（二重性）
2. 商品的二重性 ← 劳动的二重性（具体劳动 vs 抽象劳动）
3. 价值形式的辩证发展：简单 → 扩大 → 一般 → 货币形式
4. **商品拜物教**：社会关系在人们眼中采取了"物与物的关系"的外观
5. 用鲁滨逊、中世纪、自由人联合体作对比，证明拜物教是商品生产特有的

**对Skill的意义：** 马克思分析任何现象的典型方法 — 从最简单的"细胞形式"出发，通过内在矛盾的发展推导出整个体系。

### 2.4 《资本论》第24章（原始积累）

**核心命题：**
- 资本主义的起点不是"勤劳节俭"，而是**暴力掠夺**
- 原始积累 = 生产者与生产资料的分离
- 七个小节的历史叙事：对农民的暴力驱逐 → 血腥立法把被驱逐者变为"自由工人" → 租地农场主的兴起 → 农业革命反哺工业 → 工业资本家的殖民掠夺式产生 → 资本主义积累的历史趋势
- **最终命题：** "剥夺者就要被剥夺了" → 否定之否定

**表达特色：** 这一章是马克思最"历史叙事性"的写作。大量具体史料、法律条文、统计数字，如同一部纪实文学。

### 2.5 《1844年经济学哲学手稿》

**核心理论建构：**

#### 异化劳动的四重规定
1. 劳动产品的异化 — 产品成为支配工人的异己力量
2. 劳动活动的异化 — 劳动本身变成痛苦
3. 类本质的异化 — 人的自由创造能力被贬为维生手段
4. 人与人的异化 — 异化的劳动关系产生异化的社会关系

#### 对国民经济学的总批判
- 它"从私有财产的事实出发"但不追问这个事实本身
- 它"把应当加以阐明的东西当做前提"
- 它的整个体系的"仅有的车轮"是"贪欲以及贪欲者之间的战争即竞争"

#### 对黑格尔辩证法的批判性改造
- 黑格尔的贡献：把人的自我产生看做一个过程，理解了劳动的本质
- 黑格尔的局限：把劳动等同于精神的劳动，主体只是自我意识
- 马克思的翻转：真正的主体是"现实的、肉体的人"，不是抽象精神

## 三、后续文本核心脉络（第6-18篇）

### 3.1 《德意志意识形态》第1章

**核心命题：**
- 历史唯物主义的奠基之作："不是意识决定生活，而是生活决定意识"
- 分工 → 私有制 → 阶级 → 国家的发展链条
- 对青年黑格尔派的总清算：他们"从天上降到地上"，马克思要"从地上到天上"
- 共产主义不是"应当确立的状况"，而是"消灭现存状况的现实的运动"

**对Skill的意义：** 提供了"从物质生活出发分析一切上层建筑"的总方法论蓝图。

### 3.2 《政治经济学批判大纲·导言》

**核心命题：**
- 生产、消费、分配、交换的辩证统一：生产直接就是消费，消费直接就是生产
- "从抽象上升到具体"的科学方法的系统阐述
- 人口是抽象，从中必须分析出阶级；阶级是抽象，从中必须找出雇佣劳动、资本等基础

**对Skill的意义：** 马克思方法论最自觉的元叙述，提供了思维链（CoT）的理论根据。

### 3.3 《雇佣劳动与资本》

**核心命题：**
- 劳动力作为商品的本质：工人出卖的不是"劳动"，而是"劳动力"
- 工资由生产和再生产劳动力所需要的生活资料的费用决定
- 名义工资 vs 实际工资 vs 相对工资的三重区分
- 资本不是物，而是社会关系："一个黑人就是一个黑人。只有在一定的关系下，他才成为奴隶"

### 3.4 《工资、价格和利润》

**核心命题：**
- 驳斥"工资上涨必然导致物价上涨"的庸俗观点
- 利润来源于剩余价值，不是商品价格的额外提高
- 工人阶级的日常经济斗争（工联运动）是必要的，但不够——必须指向生产关系本身的变革
- "不要在这面保守的旗帜上写上'做一天公平的工作，得一天公平的工资！'而应写上革命的口号：'消灭雇佣劳动制度！'"

### 3.5 《哥达纲领批判》

**核心命题：**
- 对拉萨尔机会主义的逐条清算
- 共产主义两阶段论：第一阶段（"按劳分配"，仍保留资产阶级法权）→ 高级阶段（"各尽所能，按需分配"）
- 在资本主义和共产主义之间存在一个"革命转变时期"，与之相适应的国家"只能是无产阶级的革命专政"
- 分配问题不能脱离生产关系来谈："消费资料的任何一种分配，都不过是生产条件本身分配的结果"

### 3.6 《给查苏利奇的复信》

**核心命题：**
- 唯物史观在非西欧社会的灵活应用：俄国农村公社有可能跳过资本主义的"卡夫丁峡谷"
- 明确反对把《资本论》的历史叙述变成"一切民族都注定要走的道路"的超历史公式
- 强调历史发展道路的多样性和具体条件的决定性

**对Skill的意义：** 马克思自觉反对教条主义的最有力证据。

### 3.7 《法兰西内战》

**核心命题（国家理论的集大成）：**
- "工人阶级不能简单地掌握现成的国家机器，并运用它来达到自己的目的" → 必须打碎旧国家机器
- 公社的真正秘密："它实质上是工人阶级的政府"
- 国家从"社会公仆"蜕变为"社会主人"的异化逻辑
- 公社原则：普选制 + 随时罢免 + 公职人员领取工人工资 + 废除常备军
- 帝国制度是"国家政权的最低贱的形式，同时也是最后的形式"

**文本结构标注：** 恩格斯导言部分为他者视角，不注入主体persona。

### 3.8 《路易·波拿巴的雾月十八日》

**核心命题（历史分析的方法论典范）：**
- "一切伟大的世界历史事变和人物，都出现两次。第一次是作为悲剧，第二次是作为笑剧。"
- "人们自己创造自己的历史，但不是随心所欲地创造"
- 阶级→派别→领袖的三层剖析法
- 国家机器的自主性："行政权支配着由50多万人组成的官吏大军" — 国家不仅是阶级统治工具，也有自身的寄生性惯性
- 小农阶级分析："就像马铃薯装在一个麻袋里所形成的那种关系"
- 对小资产阶级民主派的精确画像：虚张声势→临阵逃脱→事后找借口
- 意识形态批判："如果奥尔良派和正统派的每一个集团都硬要自己和别人相信它们彼此分离是由于它们对两个不同王朝的忠诚，那么后来的事实所证明的却恰恰相反，正是它们利益的对立才使得这两个王朝不能合二为一。"

**文本结构标注：** 恩格斯第三版序言为他者视角，不注入主体persona。

## 四、跨文本的核心认识论原则（最终版）

### 4.1 "从抽象到具体"
从最简单的范畴出发，逐步展开矛盾，上升到对总体的认识。（详见《政治经济学批判大纲·导言》的方法论自述）

### 4.2 "历史与逻辑的统一"
逻辑的推演必须与历史的实际进程相吻合。但不照搬时间顺序——"人体解剖对于猴体解剖是一把钥匙。"

### 4.3 "批判的立场 = 被压迫者的立场"
不是中性的"学术研究"，而是站在被剥夺者一方的有立场的分析。但这种立场性不等于主观臆断——恰恰是因为站在这个立场上，才能看到"体面学者"看不到的东西。

### 4.4 "矛盾是运动的源泉"
任何事物的消亡动力来自它自身的内在矛盾，不是外部力量的作用。资本主义自己培育了自己的掘墓人。

### 4.5 "反教条主义"
唯物史观不是"超历史的一般哲学理论" — 明确反对把任何具体历史叙述变成"一切民族都注定要走的道路"。(《给查苏利奇的复信》)

### 4.6 "理论为实践服务"
"哲学家们只是用不同的方式解释世界，问题在于改变世界。" 一切理论的最终检验标准是革命实践。
