## 一、马克思的论战谱系

### 1.1 与古典政治经济学的论战

**对手：** 亚当·斯密、大卫·李嘉图、萨伊、穆勒

**马克思的态度分层：**
- **斯密**：既是国民经济学的"路德"（揭示了劳动是财富的主体本质），又是其局限性的典型代表
- **李嘉图**：最诚实的资产阶级经济学家——"让国民经济学用它自己的语言说话"，但恰因诚实而暴露了体系的残酷
- **萨伊**：庸俗化的代表，把复杂问题简单化
- **穆勒**：折中主义者，试图调和不可调和的矛盾

**批判策略：** 内在批判（Immanent Critique）— 采用对方的前提和范畴，从内部推出自相矛盾的结论。

### 1.2 与黑格尔哲学的论战

**双重态度：** 批判性继承

**继承了什么：**
- 辩证法作为方法 — 矛盾、否定、发展
- 历史观 — 人类通过自己的劳动自我产生
- 总体性 — 理解社会作为有机整体

**批判了什么：**
- 唯心主义 — 把观念/精神当作历史的主体
- 思辨幻想 — "自然界不过是在感性的、外在的形式下重复逻辑的抽象概念"
- 国家崇拜 — 把现存国家当作理念的实现

**经典表述：** "我的辩证方法，从根基上来说，不仅和黑格尔的辩证方法不同，而且和它截然相反。"

### 1.3 与青年黑格尔派的论战

**对手：** 施特劳斯、布鲁诺·鲍威尔

**批判要点：**
- 把宗教批判当作万能钥匙 → 只是"旧哲学的超验性被歪曲为神学漫画的顶点"
- 用"自我意识"替代"实体" → 但仍在黑格尔的框架内打转
- 宣称超越了黑格尔，实际上"甚至一点也没想到现在已经到了同自己的母亲即黑格尔辩证法批判地划清界限的时候"

**语气：** 极尽讽刺。"以批判的形式消逝着的唯心主义做出这一切滑稽可笑的动作之后……"

### 1.4 与费尔巴哈的论战

**从盟友到批判对象的转变：**
- 1844手稿：高度评价，称费尔巴哈"是唯一对黑格尔辩证法采取严肃的、批判的态度的人"
- 1845年提纲：11条全部是对费尔巴哈的批判 — "旧唯物主义的立脚点是市民社会，新唯物主义的立脚点则是人类社会或社会的人类"

**核心分歧：** 费尔巴哈把"人的本质"理解为抽象的"类"，不从社会关系和实践来理解。

### 1.5 与空想社会主义的论战

**对手：** 傅立叶、圣西门、欧文

**态度：** 尊敬但批判
- 承认他们的批判包含"最有价值的材料来启发工人"
- 批判他们：①不理解阶级斗争 ②不理解无产阶级的历史使命 ③想通过理性说服、模范社区来改变社会

### 1.6 与蒲鲁东的论战

**态度：** 从早期的某种承认到后来的尖锐批判

**批判要点：**
- 工资平等 → "只能使今天的工人对自己的劳动的关系变成一切人对劳动的关系。这时社会就被理解为抽象的资本家。"
- 利息降低 = 资本扬弃 → 马克思认为这恰恰是资本统治完成的征兆

## 二、马克思的论战方法论

### 2.1 "先入其内，再出其外"

1. **完整呈现**对手的论证 — 大段引用原文，绝不歪曲
2. **以其人之道还治其人之身** — 用对方自己的前提推出矛盾
3. **揭示盲点** — 对手看不到的东西恰恰由其立场决定

### 2.2 "追问前提"

- 对手的每一个"出发点"都被追问："你从私有财产出发，但你没有给我们说明这个事实。"
- 国民经济学家"把应当加以阐明的东西当做前提"

### 2.3 "阶级解码"

- 论战中始终追问：这个理论为谁服务？它的"客观性"掩盖了什么利益？
- 但不是简单的"阴谋论" — 而是指出理论的局限性与其阶级立场的结构性关联

### 2.4 "升级讽刺阶梯"

马克思的讽刺通常按以下阶梯递进：
1. 冷静陈述事实
2. 指出逻辑矛盾
3. 用类比/隐喻放大荒谬性
4. 终极嘲讽：把对手的结论推到极端

## 三、后续文本新增论战谱系

### 3.1 与拉萨尔机会主义的论战（《哥达纲领批判》）

**对手：** 拉萨尔及其追随者起草的合并纲领

**批判策略：** 逐句逐条批注式清算
- "劳动是一切财富的源泉" → 马克思：不对，自然界同样是源泉。这个命题"抹去了劳动的社会条件"
- "不折不扣的劳动所得" → 马克思：必须扣除扩大再生产基金、保险基金、行政管理费用等，"'不折不扣'只是表面文章"
- "铁的工资规律" → 马克思：这是拉萨尔从马尔萨斯那里借来的庸俗经济学

**语气：** 党内同志间的严厉批判——比对外论战更激烈，因为"党内的错误比敌人更危险"

### 3.2 对梯也尔及凡尔赛政府的全景式批判（《法兰西内战》）

**对手：** 梯也尔、法卢及整个资产阶级反革命

**批判策略：** 人物画像 + 事实罗列 + 反讽
- 梯也尔："这个侏儒怪物" "那帮食尸鬼组成的议会"
- 大量引用梯也尔自己的公开声明，再与其实际行动对比，形成极致反讽
- 将巴黎公社描绘为"新世界"，与凡尔赛"旧世界"形成历史性对峙

### 3.3 对小资产阶级民主派的系统批判（《雾月十八日》）

**对手：** 山岳党、社会民主派领袖

**批判策略：** 阶级分析 + 行为画像
- 民主派"总是夸大自己的力量""轻率地错误估计局势"
- "他们相信使耶利哥城墙应声倒塌的号角声的力量"
- "当他们误入死胡同时……他们就采取模棱两可的态度，尽力避免采取可能达到目的的手段"
- 逃出失败时"总是洁白无瑕"，信心比以前更坚定——"不是他们自己和他们的党应该放弃旧的观点，相反地，是形势应该来适应他们的旧观点"
