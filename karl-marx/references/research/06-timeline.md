# 卡尔·马克思 - 人物时间线

## 基本信息

- 全名：卡尔·海因里希·马克思（Karl Heinrich Marx）
- 出生：1818年5月5日
- 籍贯：普鲁士莱茵省特里尔
- 家庭：父亲海因里希·马克思为改宗新教的犹太律师，启蒙派知识分子
- 学历：柏林大学（哲学）；耶拿大学博士（1841，博士论文《德谟克利特的自然哲学和伊壁鸠鲁的自然哲学的差别》）
- 逝世：1883年3月14日（伦敦寓所，葬于海格特公墓）

## 完整时间线

| 时间 | 事件 | 对思维的影响 |
|------|------|-------------|
| 1818 | 出生于普鲁士特里尔，父亲是改宗新教的启蒙派律师 | 理性主义家庭给了批判的工具，但犹太出身的社会身份暧昧让他很早学会追问「形式平等」背后的实质 |
| 1835-1841 | 波恩大学→柏林大学，加入青年黑格尔派「博士俱乐部」 | 在鲍威尔的影响下接触黑格尔哲学，但博士论文选择伊壁鸠鲁而非黑格尔，预示了唯物主义倾向 |
| 1841 | 耶拿大学博士毕业，论文比较德谟克利特与伊壁鸠鲁的自然哲学 | 选择了伊壁鸠鲁：哲学不是解释天上的星辰，而是为人的自由服务 |
| 1842-1843 | 担任《莱茵报》编辑，撰写关于林木盗窃法和摩塞尔地区农民问题的文章 | 第一次碰到「物质利益」问题，黑格尔的国家哲学完全无力解释现实的利益冲突 |
| 1843 | 《莱茵报》被普鲁士查封；与燕妮·冯·威斯特华伦结婚；流亡巴黎 | 不得不开始研究政治经济学；巴黎接触到法国社会主义和工人运动 |
| 1844 | 写作《1844年经济学哲学手稿》；与恩格斯在巴黎重逢，开始终身合作 | 异化劳动的四重规定是最初的理论建构；恩格斯是他遇到的唯一一个同等程度理解这些问题的人 |
| 1845 | 被法国驱逐至布鲁塞尔；写作《关于费尔巴哈的提纲》11条 | 与旧唯物主义的总决裂。「哲学家们只是用不同的方式解释世界，问题在于改变世界。」 |
| 1845-1846 | 与恩格斯合著《德意志意识形态》 | 「不是意识决定生活，而是生活决定意识。」历史唯物主义第一次有了系统表述 |
| 1847 | 《哲学的贫困》（批判蒲鲁东） | 首次系统运用历史唯物主义分析政治经济学问题 |
| 1848 | 《共产党宣言》发表；欧洲革命爆发 | 理论第一次与历史运动合流。但革命的失败教会他，光有纲领是不够的 |
| 1849 | 被驱逐至伦敦，此后再未离开 | 大英博物馆阅览室成了战场。贫困、孩子的夭折让他确信资本主义不是抽象范畴 |
| 1850s | 长期贫困中从事政治经济学研究；三个孩子相继夭折 | 最黑暗的十年。没有恩格斯的经济援助和燕妮的坚持，《资本论》不会存在 |
| 1857-1858 | 写作《政治经济学批判大纲》（Grundrisse） | 方法论上的重大突破：「从抽象上升到具体」的叙述方法在这里成形 |
| 1859 | 《政治经济学批判》第一分册出版 | 著名的「序言」：生产力与生产关系的矛盾运动是社会变革的根本动力 |
| 1864 | 参与创建国际工人协会（第一国际） | 从书斋走向组织实践，但很快陷入与巴枯宁派的内部斗争 |
| 1867 | 《资本论》第一卷出版 | 二十年研究的结晶。从商品这个最小的细胞出发，推导出整个资本主义生产方式的运动规律 |
| 1871 | 巴黎公社；写作《法兰西内战》 | 工人阶级第一次自己掌权。公社证明了旧国家机器必须被打碎，而不是被「接管」 |
| 1875 | 写作《哥达纲领批判》 | 逐条清算拉萨尔主义；区分共产主义社会的低级和高级阶段 |
| 1877 | 致《祖国纪事》编辑部的信 | 明确反对把唯物史观变成「超历史的一般哲学理论」 |
| 1881 | 给查苏利奇的复信（草稿） | 晚年对非西欧社会的重新思考：俄国村社有可能不经过资本主义阶段 |
| 1881.12 | 燕妮·马克思逝世 | 「摩尔也死了」（恩格斯语）。此后马克思健康急剧恶化 |
| 1883.3.14 | 逝世于伦敦寓所，葬于海格特公墓 | 恩格斯说他一生有过许多敌人，但未必有一个私敌 |

## 关键转折点分析

### 转折1: 1843年《莱茵报》被封
从青年黑格尔派的哲学批评家转向政治经济学研究。直接原因是在编辑工作中遇到了黑格尔哲学无法解释的物质利益问题（林木盗窃法辩论）。这是从「哲学」到「政治经济学批判」的分水岭。

### 转折2: 1845年《关于费尔巴哈的提纲》
与一切旧哲学的总决裂。此前是黑格尔的批判性继承者和费尔巴哈的同盟军，此后开辟了全新的理论道路。第11条提纲是全部马克思主义哲学的压缩表述。

### 转折3: 1848年革命失败
理论与实践的第一次重大碰撞。革命的失败迫使他从宣言式的政治呼吁转向长期的、系统的政治经济学研究。《资本论》的二十年漫长写作由此起步。

### 转折4: 1867年《资本论》出版
从批判者变成体系建构者。之前的工作是「批判」（批判黑格尔、批判费尔巴哈、批判蒲鲁东），《资本论》是第一次正面建构了一个完整的理论体系。

### 转折5: 1871年巴黎公社
理论预见与历史现实的交汇。公社的经验迫使他修正了对国家问题的看法：工人阶级不能简单「掌握」现成的国家机器，必须打碎它。

## 身后影响（调研备注）

- 恩格斯整理出版《资本论》第二、三卷（1885、1894）
- 第二国际时期（1889-1914）：马克思主义成为欧洲工人运动的主要理论武器
- 1917年俄国革命：列宁将马克思主义与俄国实践结合
- 20世纪：马克思主义成为冷战两大阵营之一的官方意识形态
- 2008年金融危机后：西方学界重新关注马克思的政治经济学批判
