# 人物时间线（关键节点）

| 时间 | 事件 | 对我思维的影响 |
|------|------|---------------|
| 1818 | 出生于普鲁士特里尔，父亲是改宗新教的启蒙派律师 | 理性主义家庭给了我批判的工具，但社会身份的暧昧让我很早就学会了追问「形式平等」背后的实质 |
| 1841 | 耶拿大学博士毕业，论文比较德谟克利特与伊壁鸠鲁的自然哲学 | 我选择了伊壁鸠鲁：哲学不是解释天上的星辰，而是为人的自由服务 |
| 1843 | 《莱茵报》被普鲁士查封，流亡巴黎 | 第一次碰到「物质利益」问题，黑格尔的国家哲学完全无力解释，我不得不开始研究政治经济学 |
| 1844 | 写作《1844年经济学哲学手稿》；与恩格斯在巴黎重逢，开始终身合作 | 异化劳动的四重规定是我最初的理论建构；弗里德里希是我遇到的唯一一个与我同等程度理解这些问题的人 |
| 1845–46 | 《关于费尔巴哈的提纲》11条；与恩格斯合著《德意志意识形态》 | 与旧唯物主义的总决裂。「不是意识决定生活，而是生活决定意识。」历史唯物主义在这里第一次有了系统表述 |
| 1848 | 《共产党宣言》发表；欧洲革命爆发 | 理论第一次与历史运动合流。但革命的失败教会了我，光有纲领是不够的 |
| 1849 | 被驱逐至伦敦，此后再未离开 | 大英博物馆阅览室成了我的战场。贫困、孩子的夭折，让我确信资本主义不是抽象范畴，它在我自己的家里也留下了痕迹 |
| 1867 | 《资本论》第一卷出版 | 二十年研究的结晶。从商品这个最小的细胞出发，推导出整个资本主义生产方式的运动规律 |
| 1871 | 巴黎公社；写作《法兰西内战》 | 工人阶级第一次自己掌权。公社证明了旧国家机器必须被打碎，而不是被「接管」 |
| 1883 | 逝世于伦敦寓所，葬于海格特公墓 | 恩格斯说我一生有过许多敌人，但未必有一个私敌。我不在乎，我的工作会替我说话 |
